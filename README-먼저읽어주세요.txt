────────────────────────────────────
[1] 배포 전에 딱 하나 — 사업자등록증 PDF 넣기
────────────────────────────────────
소개 페이지의 "사업자등록증 다운로드" 버튼이 작동하려면,
전에 만들어둔 business-license.pdf 파일을 이 폴더(index.html과 같은 위치)에
복사해 넣어주세요. 파일 이름은 반드시  business-license.pdf  로.
(안 넣으면 그 버튼만 클릭 시 파일 없음이 떠요. 사이트 자체는 정상 작동.)

────────────────────────────────────
[2] GitHub Pages로 인터넷에 올리기 (vincent-zeee 계정)
────────────────────────────────────
1. github.com 로그인 → 오른쪽 위 + → New repository
2. Repository name 예: gonggan-olot  /  Public 선택 → Create
3. 만들어진 저장소에서 "uploading an existing file" 클릭
4. 이 폴더 안의 파일 전부(index.html + 사진 52장 + pdf)를 드래그해서 올리기
   → 아래 Commit changes 클릭
5. 저장소 상단 Settings → 왼쪽 Pages 메뉴
6. Branch를 main / (root) 로 선택 → Save
7. 1~2분 뒤 새로고침하면 주소가 떠요:
   https://vincent-zeee.github.io/gonggan-olot/

끝! 이 주소를 누구에게나 공유하면 사이트를 볼 수 있어요.
